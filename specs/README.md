# Specs

Feature documentation, notes, research, and design specs for the KatelyaTV project.

Files use `yyyy-mm-dd` prefix naming. When a feature is redesigned, a newer dated file supersedes the older one.

## Structure

```
specs/
  features/    Feature documentation
  notes/       Migration guides, troubleshooting, deployment notes, security reviews
  research/    Design specs and architectural decisions
```
